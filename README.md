# XeroCode Homebrew Tap

Official Homebrew tap for [XeroCode CLI](https://xerocode.ru) — chat with a team of AI models from your terminal.

## Install

```bash
brew tap SYL4R2k27/tap
brew install xerocode
```

Then sign in and use:

```bash
xerocode login
xerocode run "hello"
xerocode chat
```

## Update

```bash
brew update
brew upgrade xerocode
```

## Uninstall

```bash
brew uninstall xerocode
brew untap SYL4R2k27/tap
```

## Alternative: install via npm

If you prefer Node.js:

```bash
npm install -g xerocode-cli@beta
```
